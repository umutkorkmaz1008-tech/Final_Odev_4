# Final Değerlendirme Ödevi 4
## Graph ve Hash Table Tabanlı Lojistik Yönetim Sistemi

Bu proje, şehirler arası bağlantıları yöneten bir Graph veri yapısı ile paket sorgulamalarını hızlı şekilde gerçekleştiren Hash Table veri yapısının birlikte kullanıldığı hibrit bir lojistik yönetim sistemidir.

## Kullanılan Veri Yapıları

- Graph (Graf)
- Hash Table
- Linked List (Zincirleme)
- Queue (BFS için)

## Kullanılan Teknikler

Öğrenci numarası: 1250505068

- Hash Fonksiyonu: Tip A
- Collision Yöntemi: Zincirleme (Chaining)
- Graph Gezinme Algoritması: BFS (Breadth First Search)

## Proje Özellikleri

- Şehirler arası bağlantı oluşturma
- BFS algoritması ile graf gezme
- Paket ekleme işlemi
- Paket ID ile hızlı arama
- Hash table yapısında zincirleme collision çözümü
- Dinamik bellek yönetimi
- Bellek adreslerinin raporlanması
